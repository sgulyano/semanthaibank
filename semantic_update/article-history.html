Frame Elements (ขั้นตอนที่ 3/4)

Frame Element Description

Agent : เป็นบทบาทของผู้ที่ทำให้เกิดการกระทำที่ระบุในกริยาขึ้น โดยทั่วไปมักเป็นสิ่งมีชีวิต แต่อาจใช้กับสิ่งไม่มีชีวิตเช่นหุ่นยนต์ หรือสถาบันของมนุษย์เช่นชาติได้

Object: เป็นบทบาทสำหรับนามใดๆ ซึ่งบทบาทในการกระทำหรือในสถานะที่ระบุในกริยาขึ้นกับการตีความคำกริยา ใช้กับสิ่งที่ได้รับผลจากการกระทำหรือสถานะที่ระบุในกริยา

Manner: เป็นบทบาททางคุณภาพของเหตุการณ์ แสดงให้เห็นถึงแนวทางที่การกระทำในกริยาถูกกระทำว่าทำอย่างไร มีลักษณะอย่างไร

Measure: เป็นบทบาททางปริมาณของเหตุการณ์ แสดงให้เห็นถึงสิ่งที่เกี่ยวข้องกับเหตุการณ์ที่สามารถวัดได้ เช่นขนาด ปริมาณ ความถี่ เป็นต้น

Time:เป็นบทบาทของสิ่งแสดงเวลาที่เกิดการกระทำหรือสถานะที่ระบุในกริยา

Location: เป็นบทบาทซึ่งระบุสถานที่หรือตำแหน่งของสถานะหรือการกระทำที่ระบุในกริยา

Benefactor: เป็นบทบาทของสิ่งที่รับผลจากการกระทำ โดยที่ไม่ได้เป็นเป้าหมายของการกระทำที่ระบุในกริยาโดยตรง

Experiencer: เป็นบทบาทของสิ่งที่รับรู้การกระทำหรือสถานะที่ระบุในกริยาโดยที่ไม่ได้เป็นผู้ทำให้เกิดการกระทำหรือสถานะดังกล่าวขึ้น มักเป็นสิ่งมีชีวิต

Instrument: เป็นบทบาทของวัตถุหรือบางสิ่งที่เกี่ยวข้องกับการกระทำหรือสถานะที่ระบุในกริยา

Accompanyment: เป็นบทบาทของสิ่งที่เป็นผู้ร่วม ซึ่งอาจจะเป็นผู้ร่วมกระทำ หรือร่วมรับการแระทำก็ได้
